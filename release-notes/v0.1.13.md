## Highlights

- Settings has been reorganized with sidebar navigation, cleaner spacing, and a tidier model picker.
- Chat messages can now be copied directly from the conversation view.
- Web search is working again with the latest OpenClaw runtime.
- The desktop workspace got safer file creation, drag-and-drop imports, exports, previews, and better browser error handling.

## Stability

- Managed dev startup and proxy resolution are more reliable.
- Skill installs recover better from partial ClawHub installs, and security scan failures now expose clearer retry states.
- Desktop preview loading is throttled to keep large workspaces responsive.
- Entropic is now compatible with the latest OpenClaw runtime updates in this branch.

## Platform

- On macOS, Entropic now prefers the system Docker CLI when connecting to the local Colima runtime, avoiding hangs from the bundled client.
- Native Docker detection is more reliable on Linux.
- Windows WSL runtime recovery is more robust, and Windows release builds are now signed.
